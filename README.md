# The Next Fream of Photography

A professional photography portfolio website built with PHP and MySQL.

## Features

- Dynamic gallery with category filtering (Wedding, Portrait, Nature, Commercial)
- Contact form with database storage
- Services management from database
- Responsive design for all devices
- Email notifications for contact submissions

## Tech Stack

- PHP 7.4+
- MySQL
- HTML5/CSS3
- JavaScript
- Font Awesome
